# 💸 Expense Tracker (MERN Stack)

A full-stack Expense Tracker web application that allows users to manage their income and expenses securely with authentication and real-time updates.

---

## 🚀 Features

- 🔐 User Authentication (JWT-based login & signup)
- ➕ Add new transactions (income & expense)
- ❌ Delete transactions
- 📊 Real-time balance, income, and expense calculation
- 📈 Visual analytics using charts
- 🧑‍💻 User-specific data (each user sees only their transactions)
- 🎨 Modern UI with Tailwind CSS


## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT (Authentication)
- bcryptjs
